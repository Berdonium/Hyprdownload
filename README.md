# Hyprdownload
Hyprdownload is a free open source download manager tui that can download from github repos and other links. It is fast and optimized.
Please give me feedback to make it better

## Installation
For full installation(recommended) run :
```shell
curl -fsSL https://raw.githubusercontent.com/Berdonium/Hyprdownload/main/install.sh | bash
```

For a manuel download the github repo with :
```shell
git clone github.com/Berdonium/Hyprdownload
```

Then run :
```shell
cd Hyprdownload
```
 
And to finally install it run: 
```shell
./install.sh
```
